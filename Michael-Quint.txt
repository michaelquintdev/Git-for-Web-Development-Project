    1. What is Semantic HTML? 
        Semantic HTML is HTML that is easiest to read for both humans and machines. For instance, semantic html would be using a "section" tag as opposed to "div" since a div is 1. Harder for a human to decipher without looking at the webpage itself and 2. Isn't as specific as to what the area it surrounds is.

    2. What is HTML used for? 
        HTML is a markup language used to divide up portions of text into chunks that you can manipulate in CSS and JavaScript to do what you want.

    3. What is an attribute and where do we put it? 
        Attributes are modifier to elements. You place them after the element and are followed with typically an equals sign and its value.

    4. What is the h1 tag used for? How many times should I use it on a page?
        An h1 tag is used for the top header and should only be used once.
